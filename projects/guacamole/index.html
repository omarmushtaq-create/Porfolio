Guacamole

In this project, I set up a Guacamole server to provide remote desktop access to my virtual machines. Guacamole is an open-source remote desktop gateway that allows you to access your virtual machines from anywhere using a web browser. I installed Guacamole on a dedicated server and configured it to connect to my Proxmox virtual machines. This project helped me learn about remote desktop technologies and how to securely access virtual environments remotely.
